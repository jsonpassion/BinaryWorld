## What You'll Experience

### Mission 1 — Digital Clock
Tap a digit on a clock. Watch it break apart into 7 light segments, then discover **BCD** — the 4-bit binary code behind every digit. Toggle the bits yourself to set the time.

### Mission 2 — Text Message
A chat reveals a secret: computers can't read letters. See "Hi" decompose into **ASCII** codes, then into raw 0s and 1s. Decode a hidden 3-letter message byte by byte.

### Mission 3 — Pixel Art
A heart image pixelates step by step into an 8x8 binary grid. Each pixel is just a single bit. Complete pixel art by reading binary hints and filling in the blanks.
